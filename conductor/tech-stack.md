# Tech Stack: Flow State

## Core Technologies
- **Programming Language:** TypeScript
- **Frontend Framework:** Next.js 15 (App Router)
- **UI Library:** React 18
- **Styling:** Tailwind CSS with Radix UI primitives
- **Icons:** Lucide React
- **Device APIs:** Battery Status, Network Information, Geolocation (Haversine Geofencing)

## State & Data
- **State Management:** Zustand (with local persistence)
- **Data Fetching:** TanStack Query (with Firebase integration)
- **Database/Backend:** Firebase

## AI & Automation
- **AI Framework:** Genkit AI
- **Platform Integration:** Apple Shortcuts (via deep link webhooks with JSON payload parsing and data templating)

## Infrastructure
- **App Type:** Progressive Web App (PWA) via `next-pwa`
- **Deployment:** Vercel (inferred for Next.js)
